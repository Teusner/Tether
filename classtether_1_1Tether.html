Detailed Description

Tether class to simulate tethers using a doubly linked list of TetherElement.

Â Doubly Linked List structure

The Tether is simulated using doubly linked list. This class hold a pointer to the head and the tail TetherElement accessible through Tether::Head() and Tether::Tail().

Example

Here is a basic example of Tether use. It instanciate a Tether with some parameters, and show how to access these parameters through methods.

#include "Tether.hpp"
#include "TetherElement.hpp"
#include <cstdlib>
#include <iostream>
using namespace tether;
int main(int, char*[]) {
// Tether parameters
std::double_t length = 10;
std::size_t n = 100;
// Create a Tether
Tether tether(length, n);
// Accessing Tether parameters
std::cout << ">> Tether parameters" << std::endl;
std::cout << "n \t: " << tether.N() << std::endl;
std::cout << "length \t: " << tether.Length() << std::endl;
// Update the state from t to t + h
std::double_t h = 0.01;
tether.Step(h);
return EXIT_SUCCESS;
}
