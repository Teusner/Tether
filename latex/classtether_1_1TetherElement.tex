\doxysubsection{Detailed Description}
\mbox{\hyperlink{classtether_1_1TetherElement}{Tether\+Element}} to hold the state of a discretized tether element. 

\hypertarget{classtether_1_1TetherElement_autotoc_md7}{}\doxysubsubsection{Â \+Doubly Linked List structure}\label{classtether_1_1TetherElement_autotoc_md7}
\mbox{\hyperlink{classtether_1_1TetherElement}{Tether\+Element}} has to be used as node in a doubly linked list to simulate a \mbox{\hyperlink{classtether_1_1Tether}{Tether}}. The idea is to set the previous and the next \mbox{\hyperlink{classtether_1_1TetherElement}{Tether\+Element}} using \mbox{\hyperlink{classtether_1_1TetherElement_a8c3e7ae1d74c612966983d71b3969ad3}{Tether\+Element\+::\+Set\+Previous}} and \mbox{\hyperlink{classtether_1_1TetherElement_ab70f510c60760f453558d1ec52a0d856}{Tether\+Element\+::\+Set\+Next}}\hypertarget{classtether_1_1TetherElement_autotoc_md8}{}\doxysubsubsection{Example}\label{classtether_1_1TetherElement_autotoc_md8}
Here is a basic example of \mbox{\hyperlink{classtether_1_1TetherElement}{Tether\+Element}} use. It instanciate a \mbox{\hyperlink{classtether_1_1TetherElement}{Tether\+Element}} with some parameters, and show how to access these parameters through methods.
